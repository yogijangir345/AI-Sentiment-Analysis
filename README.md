🤖 AI Sentiment Analysis System
📌 Project Overview
The AI Sentiment Analysis System is a Machine Learning and Natural Language Processing (NLP) based web application that predicts the sentiment of a given text as Positive, Negative, or Neutral. The application provides a simple and interactive interface where users can enter any text and instantly receive sentiment predictions.

This project was developed as part of an Internship Project to demonstrate the practical implementation of Machine Learning, Text Processing, and Flask web development.

🎯 Project Objective
The primary objective of this project is to develop an intelligent sentiment analysis model capable of classifying text based on its emotional tone. The system helps analyze customer feedback, product reviews, and social media comments to understand public opinion more effectively.

🚀 Features
Predicts Positive, Negative, and Neutral sentiments
User-friendly web interface
Machine Learning-based prediction model
